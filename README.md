# Excel Data Analysis Simulation Python

This Python script simulates common data analysis tasks typically performed in Excel. It uses an in-memory dataset to demonstrate operations like calculating total revenue, aggregating sales by category, filtering data, and finding top items. The example highlights how basic programming constructs can replicate Excel's powerful analytical capabilities for understanding real-world data.

## Language

`python`

## How to Run

Save the code as `main.py`.
Run from your terminal: `python main.py`
